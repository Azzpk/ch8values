# Ch8values - 中国社会意识形态倾向测试 (Mobile)

![License](https://img.shields.io/badge/license-MIT-blue.svg)
![Platform](https://img.shields.io/badge/platform-iOS%20%7C%20Android-brightgreen.svg)

**Ch8values** 是一款专为移动端打造的中国社会意识形态倾向测试工具。它通过深度定制的问答系统，帮助用户在移动设备上快速、直观地了解自己在多个维度上的社会取向。

## 📊 测试维度
本项目从以下六个核心维度进行评估：
- **经济 (Economy)** | **社会 (Social)** | **国际 (International)**
- **组织 (Party)** | **生产 (Production)** | **文化 (Culture)**

## 📲 如何安装

### 安卓版 (Android)
1. 前往 [Releases](https://github.com/[Azzpk]/ch8values/releases)
2. 下载 `ch8values-v0.0.1.apk`。
3. 在手机设置中允许“安装未知来源应用”，即可完成安装。

### iOS 版
> **说明**：由于 iOS 平台的限制，建议通过以下方式安装：
* **越狱设备**：直接下载并安装本仓库提供的 `.ipa` 文件 前往 [Releases](https://github.com/[Azzpk]/ch8values/releases)
* **开发者自签**：下载本项目源代码，使用 Xcode 或侧载工具（如 AltStore/Sideloadly）进行安装。

## 🛠️ 技术特点
- **原生打包/WebView 驱动**：基于高性能 HTML5/Tailwind CSS 核心，确保双端体验一致。
- **离线运行**：安装后无需联网，随时随地进行测试，保护隐私。
- **动态图表**：使用 Chart.js 生成高分辨率的结果雷达图/环形图。
